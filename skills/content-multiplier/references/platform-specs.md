# Platform Specifications Reference

## Character Limits

| Platform | Limit | Ideal Length |
|----------|-------|-------------|
| X/Twitter | 280 per tweet | 200-250 per tweet |
| LinkedIn | 3,000 | 1,200-1,500 |
| Instagram caption | 2,200 | 1,000-1,500 |
| TikTok caption | 4,000 | 500-1,000 |
| YouTube description | 5,000 | 300-500 above fold |
| Email subject | ~60 (display) | Under 50 |
| Email preview | ~100 (display) | 80-100 |
| Blog title (SEO) | ~60 (SERP) | 50-60 |
| Meta description | ~155 (SERP) | 140-155 |

## Tone Guidelines

| Platform | Tone | Style |
|----------|------|-------|
| X/Twitter | Sharp, punchy | Short sentences, white space, contrarian takes |
| LinkedIn | Professional, authentic | Story-driven, vulnerable, insight-focused |
| Email | Conversational, personal | Write to one person, casual but valuable |
| Instagram | Visual-first, relatable | Emoji-friendly, hashtag-driven discovery |
| YouTube | Informative, SEO-aware | Keyword-rich, scannable, timestamp-friendly |
| Blog | Authoritative, detailed | Long-form, structured, SEO-optimized |

## Hook Formulas

1. **Contrarian**: "Stop doing X. Here's why."
2. **Story**: "We almost [bad outcome]. Here's what happened."
3. **Number**: "X lessons from doing Y for Z years"
4. **Question**: "What if everything you know about X is wrong?"
5. **Result**: "This one change increased our X by Y%"
6. **Curiosity gap**: "The #1 mistake in X (that nobody talks about)"

## CTA Templates

| Platform | CTA Type | Example |
